 = MPDAF =

== Description ==

MPDAF is the MUSE Python Data Analysis Framework.
Its goal is to develop a python framework in view of the analysis of MUSE data in the context of the GTO. 

== Installation ==

install the mpdaf package:
./setup.py build
./setup.py install

runs the test:
  ./setup.py test
